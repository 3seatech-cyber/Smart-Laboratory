# Clon estático de `evento-fia`

Este proyecto es una réplica estática de la página `https://pshiguihara.github.io/evento-fia/` para correr localmente desde VS Code.

## Abrir en VS Code

1. Abre la carpeta `evento-fia-clone` en VS Code.
2. Instala la extensión **Live Server**.
3. Haz clic derecho sobre `index.html`.
4. Selecciona **Open with Live Server**.

## Estructura

- `index.html` → estructura del sitio
- `styles.css` → estilos
- `script.js` → menú responsive
- `assets/` → logos e ilustraciones SVG locales

## Nota

Esta versión es una reconstrucción funcional para desarrollo local en VS Code. No incluye recursos binarios originales del sitio publicado.
